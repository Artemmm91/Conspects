## 7.3) Exceptions and copying

```
struct S {
	S() {}
    S(const S&) {
		std::cout << "copy\n";
	}
	~S() {
		std::cout << "destruct\n";
	}
};

void f() {
	S s;
	throw s;
}

int main() {
	try {
		f();
	} catch (S s) {
		std::cout << "caught\n";
	}
}
```

Нас интересует сколько будет копирований и разрушений. Выведется `copy destruct copy catch destruct destruct`.
